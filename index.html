Hour offset: - +

Help

Parameters:
hourOffset=2 Offsets the hour in case the browser clock is different from local time. Default offset is 9, which is the offset for me, the developer of this clock. ;-)
firstTimeDeparted=022620141137 Sets the bottom time. Format MMYYYYhhmm Note: Jan=00, Feb=01 .... Used to display time of delivery or some other date and time of significance. :-)

You can add one or multiple parameters. Order is not important. Note that first parameter must have a ? in front of it. Add more parameters by using the & sign:
?Parameter1=Value1&Parameter2=Value2

Examples:
http://tesla.ramsvik.org/?hourOffset=2&firstTimeDeparted=022620141137 (adjust hour and first departure)
http://tesla.ramsvik.org/?firstTimeDeparted=022620141137 (adjust only first departure)
